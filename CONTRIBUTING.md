# Contributing

This document outlines how to contribute to this project.

## Requirements

To develop this project, you need to have the following installed:

- [Visual Studio Code](https://code.visualstudio.com/)
- [node.js and npm](https://nodejs.org/en/)

## Getting Started

Clone this repo, and then open it up in Visual Studio Code.

Install dependencies by running `npm install` at the root of the project.

To run, use Visual Studio Code's [debug launcher](https://code.visualstudio.com/docs/editor/debugging) to launch the "Extension" configuration.

Or, you can press <kbd>F5</kbd> to launch it automatically.
